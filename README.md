# Quotation Management System

A comprehensive, role-based quotation management system built with React.js, designed for cable manufacturing companies to manage quotations, products, and sales teams efficiently.

### ðŸ‘¥ **Multi-Role Access Control**
- **Super Admin**: Full system control, user management, audit logs
- **Admin**: Manage salespersons, approve/reject quotations, product management
- **Manager**: Team management, performance monitoring, team quotations
- **Salesperson**: Create quotations, track personal performance, manage profile

### **Core Features**
- **Dynamic Quotation Creation**: Intelligent cable selection with real-time pricing
- **Product Database**: Extensive cable catalog with categories and specifications
- **Role-Based Discount Limits**: 
  - Salesperson: Up to 30% discount
  - Manager: Up to 35% discount
  - Admin/Super Admin: Up to 38% discount
- **Client Management**: Complete client information storage and tracking
- **Quotation Tracking**: Status tracking (Pending, Approved, Rejected, Won)
